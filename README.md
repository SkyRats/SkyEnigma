# SkyEnigma
Uma pessoa foi encontrada morta. As pistas estão espalhadas pelo sistema. O assassino deixou rastros — mas só quem sabe navegar pelo terminal vai encontrá-los. Sua missão: usar os comandos do Linux para vasculhar diretórios, ler arquivos suspeitos e desvendar quem cometeu o crime antes que as evidências desapareçam.
